# Bitrix24

Node.js скрипт для получения списка компаний из Bitrix24 через REST API.

## Требования

- Node.js 25.2.1
- Вебхук Bitrix24 с правами на чтение CRM (crm)

## Установка

```bash
# Клонируйте репозиторий
git clone https://github.com/BaronGalileo/Bitrix24
cd bitrix24

# Установите зависимости
npm install

#  В терминале написать 
node index.js <ваш вебхук Bitrix24>
